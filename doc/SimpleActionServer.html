Class: SimpleActionServer

SimpleActionServer

new SimpleActionServer(options)

An actionlib action server client. Emits the following events: * 'goal' - goal sent by action client * 'cancel' - action client has canceled the request
Parameters:
Name Type Description
options object with following keys: * ros - the ROSLIB.Ros connection handle * serverName - the action server name, like /fibonacci * actionName - the action message name, like 'actionlib_tutorials/FibonacciAction'
